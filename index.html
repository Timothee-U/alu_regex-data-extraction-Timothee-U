Regex Data Extraction Project

This project extracts specific types of data from large text using Regular Expressions (Regex) in Python.

Features:

  • Email Addresses
  • URLs
  • Phone Numbers (including +250 Rwandan format)
  • Credit Card Numbers
  • Time formats (12 and 24-hour)
  • HTML Tags
  • Hashtags
  • Currency Amounts
